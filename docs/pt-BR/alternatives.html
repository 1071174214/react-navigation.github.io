Alternative libraries

React Navigation não é sua única opção para roteamento e navegação em React Native. Se o passo & anti-passo ou o design da API te deixam querendo explorar outras opções, aqui estão algumas para considerar.

  • react-native-router-flux: esta biblioteca é baseada em React Navigation mas fornece uma API diferente para interagir com ele.
  • react-native-navigation: usa as APIs nativas no iOS e Android, esta é uma alternativa popular para React Navigation e vale a pena considerar se você valoriza aderir exatamente às convenções da plataforma e não importa tanto com personalização.
