createBottomTabNavigator

Uma simples barra na parte inferior da tela que permite alternar entre diferentes rotas. Rotas são inicializadas preguiçosamente - seus componentes de tela não são montados até que sejam focalizados primeiro.

Para um guia de uso completo visite Tab Navigation

createBottomTabNavigator(RouteConfigs, BottomTabNavigatorConfig);

RouteConfigs

The route configs object is a mapping from route name to a route config, which tells the navigator what to present for that route, see example from stack navigator.

BottomTabNavigatorConfig

  • initialRouteName - O nome da rota inicial a ser carregada.
  • navigationOptions - Navigation options for the navigator itself, to configure a parent navigator
  • defaultNavigationOptions - Default navigation options to use for screens
  • resetOnBlur - redefinir o estado de qualquer navegadores aninhados quando se muda para uma tela. O padrão é false.
  • order - Array de routeNames que define a ordem das guias.
  • paths - Forneça um mapeamento de routeName para path config, que substitui os caminhos definidos no routeConfigs.
  • backBehavior - initialRoute to return to initial tab, order to return to previous tab, history to return to last visited tab, or none. backBehavior - initialRoute para retornar à aba inicial, order para retornar à aba anterior, history para retornar à última aba visitada, ou none.
  • lazy - Defaults to true. If false, all tabs are rendered immediately. When true, tabs are rendered only when they are made active for the first time. Note: tabs are not re-rendered upon subsequent visits.
  • tabBarComponent - opcional, substitui o componente para usar como o tab bar.
  • tabBarOptions -Um objeto com as propriedades a seguir:
    • activeTintColor - Label and icon color of the active tab.
    • activeBackgroundColor - Background color of the active tab.
    • inactiveTintColor - Label and icon color of the inactive tab.
    • inactiveBackgroundColor - Background color of the inactive tab.
    • showLabel - Whether to show label for tab, default is true.
    • showIcon - Seja para mostrar o ícone para o tab, o padrão é verdadeiro.
    • style - Style object for the tab bar.
    • labelStyle - Style object for the tab label.
    • tabStyle - Style object for the tab.
    • allowFontScaling - Whether label font should scale to respect Text Size accessibility settings, default is true.
    • adaptive - O alinhamento dos ícones e rótulos das guias deve mudar de acordo com o tamanho da tela? O padrão é true para iOS 11. Se false, os ícones e rótulos das guias são alinhados verticalmente o tempo todo. Quando true, ícones de guia e rótulos alinham horizontalmente no tablet.
    • safeAreaInset - Override the forceInset prop for <SafeAreaView>. Defaults to { bottom: 'always', top: 'never' }. Available keys are top | bottom | left | right provided with the values 'always' | 'never'.
    • keyboardHidesTabBar - Defaults to false. If true hide the tab bar when keyboard opens.

Exemplo:

tabBarOptions: {
  activeTintColor: '#e91e63',
  labelStyle: {
    fontSize: 12,
  },
  style: {
    backgroundColor: 'blue',
  },
}

Se você desejar personalizar o tabBarComponent:

import { createBottomTabNavigator, BottomTabBar } from 'react-navigation-tabs';

const TabBarComponent = (props) => (<BottomTabBar {...props} />);

const TabScreens = createBottomTabNavigator(
  {
    tabBarComponent: props =>
      <TabBarComponent
        {...props}
        style={{ borderTopColor: '#605F60' }}
      />,
  },
);

navigationOptions para telas dentro do navegador

title

Título genérico que pode ser usado como fallback para headerTitle e tabBarLabel.

tabBarVisible

verdadeiro ou falso para mostrar ou ocultar a barra de guia, se não for definido então assume verdadeiro ``.

tabBarIcon

React Element ou uma função que forneça { focused: boolean, horizontal: boolean, tintColor: string } retorna um React.Node, para exibir na barra de guias. horizontal é true quando o dispositivo está na paisagem e false quando retrato. O ícone é renderizado novamente sempre que a orientação do dispositivo é alterada.

tabBarLabel

A sequência de títulos de uma guia exibida na barra de guias ou no React Element ou uma função que forneceu { focused: boolean, tintColor: string } retorna um React.Node, para exibir na barra de guias. Quando indefinido, a cena title é usada. Para ocultar, veja tabBarOptions.showLabel na seção anterior.

tabBarButtonComponent

React Component que encapsula o ícone e o rótulo e implementa o onPress. O padrão é um wrapper em torno de TouchableWithoutFeedback que torna se comportam da mesma como outros touchables. tabBarButtonComponent: TouchableOpacity usa TouchableOpacity.

tabBarAccessibilityLabel

Rótulo de acessibilidade para o botão do tab. Isto é lido pelo leitor de tela quando o usuário toca no tab. É recomendável para configurar isso se você não tem um rótulo para o tab.

tabBarTestID

ID para localizar este botão da guia nos testes.

tabBarOnPress

Callback para lidar com eventos de imprensa; o argumento é um objeto contendo:

  • navigation: navigation prop para a tela
  • defaultHandler: o manipulador padrão para tabulação

Útil para adicionar uma lógica personalizada antes da transição para a próxima cena (a chamada) começa.

Define this callback without ever invoking defaultHandler to prevent any navigation from occurring.

tabBarOnLongPress

Retorno de chamada para lidar com eventos de long press; o argumento é um objeto contendo:

  • navigation: navigation prop para a tela
  • defaultHandler: o manipulador padrão para tabulação
