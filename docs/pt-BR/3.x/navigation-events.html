NavigationEvents reference

NavigationEvents é um componente React que fornece uma API declarativa para assinar eventos de navegação. Ele se inscreverá em eventos de navegação no mount e cancelará a inscrição em desmontar.

Props do component

  • navigation - props de navegação (opicional, por padrão utilizará o contexto do React)
  • onWillFocus - ouvinte de eventos
  • onDidFocus - ouvinte de eventos
  • onWillBlur - ouvinte de eventos
  • onDidBlur - ouvinte de eventos

O ouvinte de evento é o mesmo que o imperativo navigation.addListener(...) API.

Example


const MyScreen = () => ( <view> <navigationevents onwillfocus={payload => console.log('will focus',payload)} ondidfocus={payload => console.log('did focus',payload)} onwillblur={payload => console.log('will blur',payload)} ondidblur={payload => console.log('did blur',payload)} /> /* Seu código do view aqui. */} </view> );

export default MyScreen; ```
