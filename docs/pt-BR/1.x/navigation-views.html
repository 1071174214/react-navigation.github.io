Vistas de navegação

Navigation views are presentation components that take a router and a navigation prop, and can display several screens, as specified by the navigation.state.

As visualizações de navegação são "controlled React Components" que podem apresentar o estado de navegação atual. Eles gerenciam a troca de telas, animações e gestos. Eles também apresentam exibições de navegação persistentes, como barras de guias e cabeçalhos.

Views feitos pelo react-navigation

  • CardStack - Apresentar uma Stack que parece adequada em qualquer plataforma
    • Card - Present one card from the card stack, with gestures
    • Header - The header view for the card stack
  • Tabs - A configurable tab switcher / pager
  • Drawer - A view with a drawer that slides from the left

Transitioner

Transitioner gerencia as animações durante a transição e pode ser usado para criar visualizações de navegação totalmente personalizadas. It is used inside the CardStack view. Learn more about Transitioner here.
