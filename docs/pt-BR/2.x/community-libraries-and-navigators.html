Navegadores

Fluid Transitions

Fluid Transitions é uma biblioteca que fornece transições de elementos compartilhados durante a navegação entre telas usando a navegação por reação.

Uma Transição de Elemento Compartilhado é a visualização de um elemento em uma tela sendo transformado em um elemento correspondente em outra tela durante a transição de navegação.

A biblioteca implementa um navegador personalizado chamado FluidNavigator que torna tudo isto e mais possível.

Links

github.com/fram-x/FluidTransitions

Bibliotecas

react-navigation-collapsible

react-navigation-collapsible é uma biblioteca e um Higher Order Component que ajusta o seu navigationOptions e torna o seu cabeçalho de tela colapsável.

Já que o cabeçalho de react-navigation foi projetado como componente Animated. Você pode animar o cabeçalho passando Animated.Value do ScrollView ou FlatList para o cabeçalho.

Links

github.com/benevbright/react-navigation-collapsible

Demo no Snack

react-native-screens

Este projeto tem como objetivo expor componentes do container de navegação nativa para React Native e o React Navigation pode se integrar a ele desde a versão 2.14.0. O uso de react-native-screens traz vários benefícios, como suporte para [ recurso de acessibilidade"](https://www.cnet.com/how-to/how-to-use-reachability-on -iphone-6-6-plus/) no iOS e melhor consumo de memória em ambas as plataformas.

Links

github.com/kmagiera/react-native-screens

react-navigation-header-buttons

Ajuda você a processar botões na barra de navegação e manipular o estilo. Ele tenta imitar a aparência dos botões nativos da barra de navegação e tenta oferecer uma interface simples para você interagir.

Links

github.com/vonovak/react-navigation-header-buttons

Demo no expo

react-navigation-props-mapper

Fornece HOCs simples que mapeiam react-navigation props diretamente para seus componentes de tela - por exemplo. em vez de const user = this.props.navigation.getParam (activeUser, null), você escreveria const user = this.props.activeUser.

Links

github.com/vonovak/react-navigation-props-mapper
