App containers

Containers são responsáveis por gerenciar o estado do seu aplicativo e vincular seu navegador de nível superior ao ambiente do aplicativo. No Android, o container do aplicativo usa a API de vinculação para lidar com o botão "Voltar". O container também pode ser configurado para manter seu estado de navegação. Em web, você usaria containers diferentes do React Native.

Nota: No v2 e anterior, os containers no React Navigation são fornecidos automaticamente pelas funções create*Navigator. A partir da v3, é necessário usar o container diretamente. No v3 também renomeamos createNavigationContainer para createAppContainer.

Um exemplo rápido de createAppContainer:

import { createAppContainer, createStackNavigator } from 'react-navigation';
// você também pode usar import from @react-navigation/native

const AppNavigator = createStackNavigator(...);

const AppContainer = createAppContainer(AppNavigator);

//Agora AppContainer é o componente principal para o React renderizar

export default AppContainer;

Props do createAppContainer em React Native

<AppContainer
  onNavigationStateChange={handleNavigationChange}
  uriPrefix="/app"
/>

onNavigationStateChange(prevState, newState, action)

Function that gets called every time navigation state managed by the navigator changes. It receives the previous state, the new state of the navigation and the action that issued state change. By default it prints state changes to the console.

uriPrefix

O prefixo dos URIs que o aplicativo pode manipular. Isso será usado ao lidar com um deep link para extrair o caminho passado para o roteador.

Chamando Despacho ou Navegar no Container do Aplicativo

No caso de você querer despachar ações em um container de aplicativo, você pode usar um React ref para chamar o método dispatch:

const AppContainer = createAppContainer(AppNavigator);

class App extends React.Component {
  someEvent() {
    // call navigate for AppNavigator here:
    this.navigator &&
      this.navigator.dispatch(
        NavigationActions.navigate({ routeName: someRouteName })
      );
  }
  render() {
    return (
      &lt;AppContainer
        ref={nav =&gt; {
          this.navigator = nav;
        }}
      />
    );
  }
}

Containers de aplicativos na Web

Na web, você pode usar createBrowserApp e handleServerRequest para manter o estado para o nível superior Navigator.
