createMaterialTopTabNavigator

Uma barra de guias com tema de design de material na parte superior da tela que permite alternar entre rotas diferentes tocando na rota ou passando horizontalmente. As transições são animadas por padrão. Componentes de tela para cada rota são montados imediatamente.

To use this navigator, ensure that you have react-navigation and its dependencies installed, then install react-navigation-tabs.

yarn add react-navigation-tabs

API

import { createMaterialTopTabNavigator } from 'react-navigation-tabs';

createMaterialTopTabNavigator(RouteConfigs, TabNavigatorConfig);

RouteConfigs

O objeto configs da rota é um mapeamento do nome da rota para uma configuração da rota.

TabNavigatorConfig

  • initialRouteName - O nome da rota inicial a ser carregada.
  • navigationOptions - Navigation options for the navigator itself, to configure a parent navigator
  • defaultNavigationOptions - Default navigation options to use for screens
  • order - Array de routeNames que define a ordem das guias.
  • paths - Forneça um mapeamento de routeName para path config, que substitui os caminhos definidos no routeConfigs.
  • backBehavior - initialRoute to return to initial tab, order to return to previous tab, history to return to last visited tab, or none. backBehavior - initialRoute para retornar à aba inicial, order para retornar à aba anterior, history para retornar à última aba visitada, ou none.
  • tabBarPosition - Posição da barra de abas, pode ser'top' ou 'bottom', o padrão é top.
  • swipeEnabled - Whether to allow swiping between tabs.
  • lazy - Defaults to false. If true, tabs are rendered only when they are made active or on peek swipe. When false, all tabs are rendered immediately.
  • lazyPlaceholderComponent - React component to render for routes that haven't been rendered yet. Receives an object containing the route as the argument. The lazy prop also needs to be enabled.
  • initialLayout - Optional object containing the initial height and width, can be passed to prevent the one frame delay in react-native-tab-view rendering.
  • tabBarComponent - Opcional, substitua o componente a ser usado como a barra de guias.
  • tabBarOptions - An object with the following properties:
    • activeTintColor - Label and icon color of the active tab.
    • inactiveTintColor - Label and icon color of the inactive tab.
    • showIcon - Whether to show icon for tab, default is false.
    • showLabel - Whether to show label for tab, default is true.
    • upperCaseLabel - Whether to make label uppercase, default is true.
    • pressColor - Color for material ripple (Android >= 5.0 only).
    • pressOpacity - Opacity for pressed tab (iOS and Android < 5.0 only).
    • scrollEnabled - Whether to enable scrollable tabs.
    • tabStyle - Style object for the tab.
    • indicatorStyle - Style object for the tab indicator (line at the bottom of the tab).
    • labelStyle - Style object for the tab label.
    • iconStyle - Style object for the tab icon.
    • style - Style object for the tab bar.
    • allowFontScaling - Whether label font should scale to respect Text Size accessibility settings, default is true.
    • renderIndicator - Função que pega um objeto com a rota atual e retorna um Elemento personalizado para ser usado como um indicador da aba.

Exemplo:

tabBarOptions: {
  labelStyle: {
    fontSize: 12,
  },
  tabStyle: {
    width: 100,
  },
  style: {
    backgroundColor: 'blue',
  },
}

navigationOptions para telas dentro do navegador

title

Título genérico que pode ser usado como fallback para headerTitle e tabBarLabel.

swipeEnabled

True ou False para ativar ou desativar o deslizamento entre guias, se não estiver definido, o padrão é a opção TabNavigatorConfig swipeEnabled.

tabBarIcon

React Element ou uma função que forneça { focused: boolean, horizontal: boolean, tintColor: string } retorna um React.Node, para exibir na barra de guias. horizontal é true quando o dispositivo está na paisagem e false quando retrato. O ícone é renderizado novamente sempre que a orientação do dispositivo é alterada.

tabBarLabel

A sequência de títulos de uma guia exibida na barra de guias ou no React Element ou uma função que forneceu { focused: boolean, tintColor: string } retorna um React.Node, para exibir na barra de guias. Quando indefinido, a cena title é usada. Para ocultar, veja tabBarOptions.showLabel na seção anterior.

tabBarAccessibilityLabel

Rótulo de acessibilidade para o botão do tab. Isto é lido pelo leitor de tela quando o usuário toca no tab. É recomendável para configurar isso se você não tem um rótulo para o tab.

tabBarTestID

ID para localizar este botão da guia nos testes.

tabBarOnPress

Callback para lidar com eventos de imprensa; o argumento é um objeto contendo:

  • navigation: navigation prop para a tela
  • defaultHandler: o manipulador padrão para tabulação

Útil para adicionar uma lógica personalizada antes da transição para a próxima cena (a chamada) começa. Ao definir tabBarOnPress, o defaultHandler precisa ser chamado para executar a ação padrão (ou seja, a switch tab).

tabBarOnLongPress

Retorno de chamada para lidar com eventos de long press; o argumento é um objeto contendo:

  • navigation: navigation prop para a tela
  • defaultHandler: o manipulador padrão para tabulação
