导航视图

导航视图是采用 路由 和 navigation prop ，并且可以显示多个页面，如navigation.state所指定的。

导航视图是可以呈现当前 navigation state 的受控 React 组件。 他们管理页面、动画和手势的切换。 他们还呈现持久导航视图，如标签栏和标题栏。

内置视图

  • StackView - 呈现一个适合任何平台的堆栈
    • StackViewCard - 从 Card堆栈中呈现一张 Card， 并带有手势操作
    • Header - Card 堆栈的标题栏视图
  • SwitchView - A navigator that only ever show one screen at a time, useful for authentication flows.
  • Tabs - 可配置的 tab switcher / pager
  • Drawer - 从左边滑入的带抽屉的视图

Transitioner

Transitioner管理页面切换期间的动画，可用于构建完全自定义的导航视图。 它在CardStack视图中使用。 了解更多关于 Transitioner 的信息
