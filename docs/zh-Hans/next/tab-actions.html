TabActions reference

TabActions is an object containing methods for generating actions specific to tab-based navigators. Its methods expand upon the actions available in NavigationActions.

支持以下操作：

jumpTo

The jumpTo action can be used to jump to an existing route in the tab navigator.

  • name - string - Name of the route to jump to.
  • params - object - Screen params to merge into the destination route (found in the pushed screen through route.params).
import { TabActions } from '@react-navigation/routers';

const jumpToAction = TabActions.jumpTo('Profile', { name: 'Satya' });

navigation.dispatch(jumpToAction);
